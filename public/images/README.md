# Images Directory

This directory contains all the images used in the Rentals landing page.

## Structure

- `/properties/` - Property listing images
- `/testimonials/` - Customer testimonial profile images
- `/icons/` - App icons and favicons

## Image Guidelines

- Use WebP format when possible for better compression
- Optimize images before adding them to the project
- Keep file sizes reasonable (< 500KB for most images)
- Use descriptive filenames

## Placeholder Images

Currently, the project uses icon placeholders. Replace them with actual property images before deployment.
